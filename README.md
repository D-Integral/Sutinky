# Usage

The library is available in Swift Package Manager.

1)
File -> Add Package Dependencies... -> https://github.com/D-Integral/Sutinky.git

2)
<code>import Sutinky</code>

3)
<code>
var body: some View {
    SutinkyCalendarView()
}
</code>
